# 🧠 QUIZ ARENA

A futuristic and responsive quiz web application designed with a premium glassmorphic interface, interactive gameplay, category-based challenges, and real-time score tracking.

## 🚀 Project Overview

QUIZ ARENA is an interactive multiple-choice quiz application that allows users to test their knowledge through category-based challenges while tracking progress, scores, and performance in a modern gaming-inspired interface.

Developed as part of **SkillCraft Technology Internship – Task 03**.

## ✨ Features

* Multiple Choice Quiz System
* Category Selection (HTML, CSS, JavaScript)
* Real-Time Score Tracking
* Countdown Timer for Each Question
* Progress Bar Indicator
* High Score Storage using Local Storage
* Player Name Personalization
* Interactive Answer Validation
* Performance-Based Result Screen
* Glassmorphism UI Design
* Responsive Layout
* Smooth Animations and Hover Effects
* Futuristic Quiz Dashboard Theme

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
